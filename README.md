毫米波感知——非接触式毫米波雷达生命体征智能监护系统
一、项目简介
本项目基于毫米波雷达技术，开发了一套面向养老机构、医院ICU及工业安监场景的非接触式生命体征智能监护平台。系统由个人端、企业端双页组成，实现了实时心率、呼吸频率、睡眠分期等生理指标的可视化监测，同时具备跌倒异常预警、数据统计与趋势分析功能，为用户提供安全、便捷的远程监护服务。

二、 项目文件结构
health-monitor/
├── index.html        # 项目首页/登录页
├── personal.html     # 个人用户监护端页面
├── enterprise.html   # 企业/机构管理端页面
└── bj.jpg            # 页面背景图片

三、核心功能
1.  非接触式生命体征监测：通过毫米波雷达技术，无接触采集并实时展示心率、呼吸频率数据。
2.  睡眠分期可视化：采用河流堆叠图形式，直观呈现用户深睡、浅睡、快速眼动睡眠阶段分布。
3.  异常跌倒预警：内置跌倒检测算法，触发异常时自动声光报警并记录日志。
4.  双端数据管理：个人端查看自身健康数据，企业端实现多用户监护、数据汇总与趋势分析。
5.  响应式界面设计：适配多端设备，支持深色科技风主题，交互友好。

四、部署方式（GitHub Pages）
1.  将项目文件上传至GitHub仓库（确保所有文件位于仓库根目录）。
2.  进入仓库 `Settings` → `Pages`。
3.  选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/root`。
4.  保存设置，等待部署完成后，即可通过生成的永久网址访问项目。

五、 技术栈
前端：HTML5 + CSS3 + JavaScript
可视化图表：ECharts
部署平台：GitHub Pages
设计风格：科技风深色主题，适配多端响应式布局


六、 使用说明
1.  打开项目首页 `index.html`，可跳转至个人端或企业端。
2.  个人端：实时查看自身生命体征数据、睡眠报告及历史趋势。
3.  企业端：管理多个用户监护数据，查看汇总统计报表与预警记录。
4.  点击界面按钮可手动触发跌倒检测、刷新数据或切换主题模式。
